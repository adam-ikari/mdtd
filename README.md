# Markdown TODO (mdtd)

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

A command-line TUI (Text User Interface) todo tool that stores tasks in Markdown format.

## Features

- Simple and intuitive TUI interface
- Tasks stored in standard Markdown format
- Cross-platform support
- Lightweight and fast

## Environment Dependencies

- uv package manager

## Usage

```bash
uv run main.py [file name]
```
