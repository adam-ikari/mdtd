# Markdown TODO (mdtd)

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

一个命令行 TUI (文本用户界面) 待办事项工具，使用标准 Markdown 格式存储任务。

## 功能特性

- 简单直观的 TUI 界面
- 任务以标准 Markdown 格式存储
- 跨平台支持
- 轻量快速

## 环境依赖

- uv 包管理器

## 使用方法

```bash
uv run main.py [文件名]
```
